# Springboot User Registration Email System

Functional backend system that allows a user to be created via email verification.
Verification link sent to email expires in 15 minutes.
Postgres backend
Spring Boot Dependencies:

- Spring Web
- Spring Security
- Postgre SQL Driver
- Spring Data JPA
- Java Mail Sender
- Lombok
